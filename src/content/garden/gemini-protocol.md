## What is Gemini?
gemini:// is a web protocol designed for uncluttered text based websites and user privacy, established in June 2019. A more modernized/streamlined version of [Gopher](https://en.wikipedia.org/wiki/Gopher_(protocol)). Essentially a text-only Internet.

## Some ways to use Gemini
- [Geminaut](https://www.marmaladefoo.com/pages/geminaut) - user-friendly UI client (Windows only)
- [Lagrange](https://github.com/skyjake/lagrange) - another UI client ([setup tutorial for Linux](https://fossforce.com/2026/03/exploring-geminispace-with-lagrange-on-linux/))
- [Amfora](https://github.com/makew0rld/amfora) - terminal based client
- [Smolnet Portal](https://portal.mozz.us/) - proxy service to access gemini:// servers in a normal web browser
